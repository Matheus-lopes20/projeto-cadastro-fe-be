# Projeto de Cadastro

Estrutura:
- backend: Node.js + Express + SQLite
- frontend: React + axios

## Como rodar

1. Backend
   ```
   cd backend
   npm install
   npm start
   ```
2. Frontend
   ```
   cd frontend
   npm install
   npm start
   ```
3. Acesse http://localhost:3000

Observações:
- O banco SQLite (`cadastro.db`) será criado automaticamente na pasta `backend` ao iniciar o servidor.
